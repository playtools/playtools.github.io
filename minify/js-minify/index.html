            How to use:
        
    
            
  • Paste your JavaScript code in the Paste JS Code box OR enter a direct JS file URL in the
                Or enter JS file URL box.

  • Click the Minify Now button below.
    •

        Sample JS:
        
// Example JS
function hello(name) {
    // Greet user
    console.log("Hello, " + name + "!");
}
hello("World");
